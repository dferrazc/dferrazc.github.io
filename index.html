FCG Class
Aug 2015 - Sep 2015

FCG is an abbreviation for Full Customized GUI, a class that I developed to ease the process of fuly customizing the User Interface of my apps in Windows Desktop, for accomplish that it was necessary to stop windows from automatic drawing the standard GUI and then drawing it on my own with DirectX libraries but still letting the system handle some messages for viability's reasons. It's based on Win32 and because of that the app stays light, smooth and customized.
